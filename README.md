# LOJO

Браузерная божья песочница в духе WorldBox: рисуй биомы, призывай людей и зверей, обрушивай катастрофы — а поселенцы сами добывают, строят и выживают.

## Играть онлайн

**https://eldar-p.github.io/lojo/**

### Один раз включить GitHub Pages

У агента нет прав admin на репозиторий, поэтому Pages нужно включить владельцу:

1. Открой [Settings → Pages](https://github.com/eldar-p/lojo/settings/pages)
2. **Build and deployment → Source** → выбери **GitHub Actions**
3. Дождись зелёного workflow [Deploy to GitHub Pages](https://github.com/eldar-p/lojo/actions/workflows/deploy-pages.yml)  
   (если не стартовал сам: Actions → Deploy to GitHub Pages → Run workflow)

После этого игра будет открываться по ссылке выше.

## Локальный запуск

Нужен простой локальный сервер (ES-модули не открываются через `file://`):

```bash
python3 -m http.server 8080
```

Открой [http://localhost:8080](http://localhost:8080).

## Как играть

- Внизу силы бога: **Мир** / **Жизнь** / **Хаос** / **Поселение**
- Зажми **ЛКМ** и веди кистью — рисуй воду, лаву, лес, снег
- Кликай молнии, метеориты, торнадо, бомбы
- Призывай людей, кроликов, волков и бандитов
- **ПКМ** — камера, **WASD** — панорама, **1/2/3** — размер кисти
- Вкладка **Поселение** — стройка и добыча, как раньше

## Война и стратегии

Вкладка **Война**:
- Постройки: стена, ворота, башня, казарма
- Доктрины как в реальной тактике: оборона, рейд, охват, осада, прорыв, засада
- Орда выбирает стратегию по силам и вашим укреплениям
- Башни стреляют, стены ломают в осаде, солдаты учатся в казарме

## ИИ (локальный, без API)

Поселенцы думают через **Utility AI** в `js/brain.js`:
- роли: строитель / собиратель / охотник / страж
- оценка целей: еда, сон, бегство, бой, работа, охота, патруль
- A* с кучей и штрафами за огонь/лаву
- волки ходят стаей и заходят с флангов, бандиты грабят склад

Никаких сетевых LLM/API — только JS в браузере.

## Стек

Чистый HTML / CSS / Canvas / ES-модули — без сборщика.
